Modelado de Legos
Para un curso de Modelado 3D, se nos dio como encargo modelar una pieza de Lego de nuestra elección, usando medidas reales. Yo escogí la pieza de 4x2 plana y gruesa para este encargo. Usando piezas modeladas por otros estudiantes, se nos dio como tarea construir una figura de Lego. En mi caso, decidí construir una nave con las piezas que había a disposición.
Modelado de una Botella
Para otro encargo, cada estudiante escogió una botella de agua para modelar usando las medidas de la botella o también usando una imagen como referencia. Se debió modelar la botella, la tapa y el plástico que cubre estas botellas.
Modelado de un objeto
Para otro curso, tuvimos que modelar un objeto de escritorio, algo que no superase los 15 cm y luego construirlo usando corte laser. Modelé un soporte para mi consola que tiene aspecto de mini arcade para este ejercicio.
